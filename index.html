Ejercicio 1

Instrucciones

Se tiene el siguiente arreglo:
let arr = ["This", "is", "a", "sentence."];
Crear una función : function printOutString(arr) de tal modo que la salida sea: This is a sentence.
Sugerencia: Usar join().

Ejercicio 2

Instrucciones

Se tiene el siguiente arreglo:
let arr = [1, 2, 4, 5];
Crear una función que duplique cada elemento del arreglo, de tal modo que la salida sea: [2, 4, 8, 10]
Sugerencia: Usar map().

Ejercicio 3

Instrucciones

Elaborar una función que calcule tanto la suma, como el producto de un arreglo.
Ejemplo: Se tiene el arreglo let array = [1, 2, 3, 4];
La salida debe ser: La suma es 10, el producto es 24.
Sugerencia: Usar reduce().

Ejercicio 4

Instrucciones

Se tienen dos arreglos con una serie de cursos:
let student1Courses = ['Math', 'English', 'Programming'];
let student2Courses = ['Geography', 'Spanish', 'Programming'];
Elaborar una función que imprima los cursos en común
Sugerencia: Usar filter() e includes().

Ejercicio 5

Instrucciones

Se tiene el siguiente arreglo: let people = ["Maria", "Dani", "Luis", "Juan", "Camila"];
Elaborar una función que:

  1. Imprima la lista completa.
  2. Elimine "Dani" del arreglo.
  3. Elimine "Juan" del arreglo.
  4. Coloque "Luis" en la primera posición.
  5. Agregue tu nombre al final del arreglo.
  6. Itere por el arreglo hasta encontrar "Maria".
  7. Imprima el índice de "Maria" en el arreglo.
  8. Imprima la lista final de 4 personas
